# Barra de Treino Shopee Landing Page

Landing page responsiva para venda de barra de treino ajustável com direcionamento para compra na Shopee.

## Arquivos principais

- `index.html` - estrutura da página
- `styles.css` - identidade visual, responsividade e efeitos 3D
- `script.js` - interações do hero e cards
- `assets/` - imagens e vídeos usados na landing page

## Como abrir localmente

Abra o arquivo `index.html` no navegador ou rode um servidor local:

```powershell
& "C:\Users\scoff\AppData\Local\Programs\Python\Python314\python.exe" -m http.server 8000 --bind 127.0.0.1
```

Depois acesse:

```text
http://127.0.0.1:8000
```
